# WeIP Messenger Lite

WeIP Messenger Lite is a lightweight Windows messaging application designed for fast, simple, and efficient communication.

## Features
- ⚡ Lightweight and fast
- 💬 Real-time messaging
- 📁 File sharing
- 🖥️ Optimized for Windows
- 🌐 Low internet usage
